READMEINSTRUCTIONS TO EXECUTE PCA.pyDEPENDENCIES:
Python     3.6.1
Pandas     0.20.1
Numpy      1.13.1
Matplotlib 2.0.1

EXECUTION IN COMMAND LINE:
 python3 PCA.py [filename]

Example:
 python3 PCA.py example.txt
 

To execute the PCA.py, provide the complete path of the .txt file as the first command line argument.
